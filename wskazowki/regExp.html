<--RegExp Wyrażenia regularne

Wyrażenia regularne to schemat służący do opisu i wyszukiwania konkretnego ciągu znaków w tekście.
Zdecydowanie więcej przeczytasz tutaj:
https://miroslawmamczur.pl/wyrazenia-regularne-czym-sa-i-jak-pisac-wlasne-regexy/


KOD

//tworzymy wyrażenie regularne
//g: przeszukuje cały element nie zatrzymując się na pierwszym dopasowaniu
//i: znajdzie element bez względu na wielkość liter
let regexp = new RegExp("rower", "gi");
//tak wyszukuje się elementy
//spawdzamy tablice obiektów
let newTab = cities.filter((findElement) => {
//obiekt posiada element "city"
//funkcja match pozwala odnaleźć elementy które pasują do wyrażenia regularnego
return findElement.city.match(regexp);
});
