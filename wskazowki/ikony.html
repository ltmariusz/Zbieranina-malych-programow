<--ikony

Tworząc strony internetowe chcemy dodać różnego rodzaju ikony, obrazki, symbole. W takich sytuacjach możemy skorzystać np. z jednej z darmowych bibliotek jest: "Font Awesome"

KOD

<style>

// Inicjowanie zmiennej
<script src="https://kit.fontawesome.com/c9ebcf1db5.js" crossorigin="anonymous" ></script>


// Wykorzystanie ikony
p{
<i class="fa-solid fa-house"></i>
}

</style>
