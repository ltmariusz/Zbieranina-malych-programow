<-- Jak ostylować hiperłącza

Standardowo hiperłącza mają wbudowane ostylowanie. Cztery podstawowe stany to:

  • a:link - normalny nie odwiedzony link
  • a:visited - link, który użytkownik odwiedził
  • a:hover - link, gdy użytkownik najedzie na niego myszką
  • a:active - link w momencie kliknięcia

UWAGA!

Podczas ustawiania stylu dla kilku stanów łącza obowiązują pewne zasady dotyczące kolejności:

  • a:hover MUSI następować po a:link i a:visited
  • a:active MUSI wystąpić po a:hover

KOD

/* #cwiczenie oznacza że ostylowany został znacznik 'a' o 'id' równym 'cwiczenie', ten link znajduje się pod przykładowym kodem
*/

a:link#cwiczenie {
color: red;
}

a:visited#cwiczenie {
color: green;
}

a:hover#cwiczenie {
color: hotpink;
}

a:active#cwiczenie {
color: rgb(255, 255, 255);
}
