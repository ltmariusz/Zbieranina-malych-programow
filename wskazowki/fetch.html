<--fetch (pobieranie danych z bazydanych)

fetch() to funkcja która pozwala pobierać elementy z baz danych w javascript. W funkcji wpisujemy adres z którego mamy pobrać elementy. Nastęnie można zrobić jeszcze wiele rzeczy (in progress)


KOD

fetch(endpoint).then((respone) => {
if (respone.ok) {
respone.json().then((data) => {
console.log(data);
});
}
});
