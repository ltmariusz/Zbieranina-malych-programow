<--Zmienne CSS

Zmienne pozwalają przechowywać pewne informacje, które możemy wykorzystać poźniejszym czasie, odnosić się do nich. Dobrym pomysłem jest zapisywanie kolorów, wielkości np. czcionek, grubości ramek itd.

KOD

<style>

// Inicjowanie zmiennej
:root {
--textSize: 80px;
}

// Wykorzystanie zmiennej
p{
font-size: var(---textSize)
}

</style>
